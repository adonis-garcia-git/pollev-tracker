Privacy Policy – PollEv Attendance Notifier

PollEv Attendance Notifier is a Chrome extension that monitors Poll Everywhere (PollEv) pages for active attendance polls during the schedule configured by the user and sends notifications.

Data Stored Locally
The extension stores PollEv usernames, class schedules (days and times), optional class end dates, and notification preferences using Chrome storage on the user’s device.

Data Transmitted
The extension loads PollEv pages configured by the user to detect active polls.
If phone notifications are enabled, the extension sends a notification message to ntfy (https://ntfy.sh) using the topic provided by the user. Messages may include a class name and a brief poll alert.

Data Not Collected
The extension does not collect passwords, authentication credentials, financial data, health data, browsing history outside PollEv, or advertising identifiers.

Data Retention
All stored data remains until the user edits settings or uninstalls the extension.

Limited Use
User data is used solely to provide the extension’s core functionality and is not sold, shared for advertising, or used for unrelated purposes.
